# Translate Api

A brief description of what this project does and who it's for


## Usage

```http
  POST https://translate-api-man.vercel.app
```

| Body      | Type     | Description                |
| :-------- | :------- | :------------------------- |
| text      | `string` | The text you want to translate |
| language  | `string` | Selected language |
